# Ecommerce_Data_Analysis
End-to-end Ecommerce Data Analysis project using Python, SQL, Excel and Tableau. Covers data cleaning, missing value treatment, customer demographics, product performance, payment behavior, sentiment insights, and dashboard visualizations for strategic decision-making.

This project demonstrates a complete data analytics workflow using a real-world ecommerce dataset containing 306,000+ customer transactions.

Loading and exploring data in Python

Performing EDA & data cleaning

Running SQL Server queries

Building an Excel dashboard

Creating an interactive Tableau dashboard

Preparing a detailed report

Creating a presentation using Gamma
